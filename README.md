#  React-meteo

### Описание проекта:
В данном проекте создано одностраничное приложение - Weather. Реализована возможность добавления, удаления текущих прогнозов.

### Публикация:
https://alsu-dxdy.github.io/react-meteo/

### Стек проекта:
- React
- Redux
- JS

### Планы по развитию проекта:
Добавить возможность сохранения прогнозов в localstorage, 
добавить иконки погодных условий,
перенести подключение функции connect из App в отдельный контейнер.

### Чтобы начать работу с проектом:

- Клонировать репозиторий:

```
git clone https://github.com/alsu-dxdy/react-meteo
```

- Установить зависимости: в папке проекта в терминале ввести:

```
npm install
```

- Запустить проект: в папке проекта в терминале ввести:

```
npm start
```
- Enjoy!
